This study explores an ensemble learning approach for classifying skin lesions using the HAM10000 dataset. A number of widely used deep learning models such as EfficientNet, Dense- Net, several ResNeXt variants, InceptionV3, and swAV were tested under five-fold cross-valida- tion and compared with the proposed ensemble. The ensemble consistently came out on top, achieving a mean accuracy of 0.8622 along with notable improvements in macro-F1, precision– recall, and ROC-AUC scores. These results indicate that combining multiple architectures leads to a more stable and balanced classifier than relying on any single model, especially given the uneven class distribution in skin lesion data. One limitation of this work is that all experiments were carried out on a single dataset. HAM10000 is dominated by images from fair-skinned patients, which raises questions about how well the model would perform on darker skin tones or in different clinical environments. To better understand the method’s generalizability, future studies should test the ensemble on other public datasets such as BCN20000 and PH², and ideally on more diverse collections overall.
